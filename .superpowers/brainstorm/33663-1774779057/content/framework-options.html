Which framework should we use?

Both support MDX and modern UI, but they differ in performance and complexity.

A

Astro (Recommended)

Zero-JS by default for industry-leading performance. Built specifically for content sites. Great MDX support and easy deployment.

  • Fastest performance
  • Simpler mental model
  • Smaller ecosystem than Next.js
B

Next.js (App Router)

The industry standard for React apps. Powerful but more complex. Best if you plan to add highly interactive apps or dashboards later.

  • Huge ecosystem (Shadcn UI)
  • Powerful dynamic features
  • Higher JS overhead
  • More complex build/config
